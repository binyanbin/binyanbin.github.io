linux基本概念

2016年11月15日

用户

sudo 超级管理员权限
groups 用户组 共享一些资源和权限

权限管理

查看文件权限 ls -l
修改权限
chmod u拥有者g群组o其它a所有 +加-减=设定 r读w写x执行
示例：chmod u=rwx,g+rx,o-x test.sh

目录结构

boot 开机相关
etc 配置文件
home 用户目录
mnt 挂载其它
opt 附加软件包
root 超级管理员目录
sbin 超组管理员的命令
bin 一般用户的命令
usr 用户于存储用户只读数据
var 不断变化的文件如日志邮件

环境变量

全局环境变量 export test =”aa”
局部环境变量 test =”aa”
环境变量存在内存中，重启后消失。
如果想开机使用:1)所有用户使用 /etc/profile export变量
2)当前用户使用 ~/.bash_profile 或~/.bashrc

管道

前面每一个进程的输出(stdout)直接作为下一个进程的输入(stdin)
ps -ef |grep nginx

数据流重定向

重定向有两种方式，一种是直接输出，使用一个大于号“>”实现；另一种是以附加的方式输出，使用两个大于号“>>”实现。前者会覆盖原始的输出内容，而后者会添加到文件最后。
